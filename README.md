# 🎮 Pixel Portfolio Website

A retro-inspired personal portfolio website built using **HTML, CSS, and JavaScript**. This portfolio showcases my projects, technical skills, education, internship experience, certifications, and contact information through an interactive pixel-art themed interface.

The website combines modern web development with nostalgic game-inspired visuals, including custom animations, a playable mini-game, smooth transitions, dark mode, and interactive project cards.

---

## 📌 Features

- 🎨 Retro pixel-inspired UI design
- 🌙 Light and Dark mode
- 🖱️ Custom pixel cursor
- ⚡ Animated loading screen
- ⌨️ Typing text animation
- 📜 Smooth scrolling navigation
- 🎮 Playable Crab Runner mini-game
- 📱 Fully responsive layout
- 💼 Interactive project showcase
- 🧠 Skills categorized by domain
- 🎓 Education and certifications section
- 👨‍💻 Internship experience
- 📞 Contact section
- ✨ Animated background effects
